Some Things I Built

01

Railway Reservation System

The Railway Reservation System is a backend RESTful API application built using Java and Spring Boot, designed to manage train schedules, passenger information, and ticket reservations. It provides endpoints for creating and managing trains, passengers, and bookings, including features like route-based train search, seat availability checks, and reservation status updates. This project demonstrates core concepts of Spring MVC, JPA (Hibernate), and REST architecture, making it a solid foundation for building a scalable transportation booking system.

02

Course and Banking Management System

This Spring Boot project is a Course and Banking Management System that provides a RESTful API to manage course-related data, user accounts, and transaction histories. It allows users to add, retrieve, update, and delete course details, apply filters based on price and offers, and search courses by name or stream. Additionally, it offers core banking features such as account creation, deposits, withdrawals, balance checks, and detailed transaction histories with time and amount filters. The system is modular, scalable, and ideal for learning and implementing CRUD operations, REST APIs, and backend service architecture with Spring Boot.

03

Food Store Web Application

A responsive web application that allows users to browse a diverse food menu, place online orders, and enjoy features like live order tracking, discounts, and multiple payment options. Designed to offer a smooth and modern online food shopping experience.
